01

The Promise Is Real. The Plumbing Is a Nightmare.

Ready 312 words · edited 4 min ago

Every CTO has seen the demo. Three lines of LangChain, a vector store, a smiling chatbot that actually answers the question. The room nods. The pilot ships. And then somebody asks the question that quietly torpedoes the project: how do we run this in production?

The honest answer, in most enterprises, is we built a one-off. A Python script on an EC2 box, a Pinecone key in a Slack DM, a model API call wrapped in retry logic written by an intern who has since left.

02

What an Agentic App Actually Needs From Its Platform

Ready 287 words

Set aside the model for a moment. The model is the easy part — somebody else makes it, you pay for tokens, you call an API. What's hard is everything around it.

An agentic app needs identity. It needs network policy. It needs secrets that don't live in dotenv files. It needs observability for prompts and tool calls, not just CPU graphs.
